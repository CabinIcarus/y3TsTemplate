# y3TsTemplate

使用更现代的TS来开发, 速度++,安全++

使用方式:
1. 安装
- - 直接使用github的模板创建功能,也就是 在克隆附近的绿色大按钮 `Use this template`
- - 直接克隆仓库
2. 修改`ProjectCopnfig.js`中的`Y3ProjectPath`为创建好的项目所在路径
3. 控制台运行 `yarn install`
4. 开始写代码
